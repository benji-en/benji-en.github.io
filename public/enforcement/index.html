You can prefix your search with "~" to search with multiple individual words/phrases.

Separate each word/phrase with a ";"

Example: "~shooting;gun" will show all rules that contain the words "shooting" or "gun"

"~base location;one" will show all rules that contain the phrase "base location" or the word "one"


URL Parameters:


  • ?search=
  • Will fill the search box with whatever the parameter is set to.
  • This parameter is synced with the contents of the search box.
  • Examples:
  • "?search=shooting%20your%20gun" (some characters need to be URL encoded)
  • "?search=~shooting;gun" (searching using the filtering explained above)
  • ?rule=
  • Will jump to the provided rule "acronym" (A10, EMISC1, GLR2, etc)
  • Examples:
  • "?rule=ekr1"
  • "?rule=ger1"
  • "?rule=a11"

These URL parameters are mutually exclusive, which means you can only use one at a time.
